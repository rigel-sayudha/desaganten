1. PERSIAPAN AWAL
================================================================================

PERSYARATAN SISTEM:
- Windows 10/11 (64-bit) atau Linux/macOS
- RAM minimum 4GB (disarankan 8GB untuk Chart 3D)
- Storage kosong minimal 5GB (bertambah karena fitur Chart.js 3D)
- Koneksi internet untuk download package dan CDN Chart.js
- Browser modern (Chrome, Firefox, Safari) untuk fitur 3D

PERSIAPAN FOLDER:
1. Buat folder di C:\ (Windows) atau /var/www/ (Linux)
2. Nama folder: xampp (Windows) atau html (Linux)
3. Path lengkap: C:\xampp\htdocs\ (Windows) atau /var/www/html/ (Linux)

TEKNOLOGI YANG DIGUNAKAN:
- Laravel 10.x (Backend Framework)
- Chart.js + chartjs-plugin-3d (3D Charts)

8. KONFIGURASI DATABASE
================================================================================

BUAT DATABASE:
1. Buka browser
2. Ketik: http://localhost/phpmyadmin
3. Tekan Enter
4. Login dengan:
   - Username: root
   - Password: (kosong/blank)
5. Klik tab "Databases"
6. Ketik nama database: desaganten
7. Pilih Collation: utf8mb4_unicode_ci
8. Klik "Create"

EDIT FILE KONFIGURASI:
1. Buka file: C:\xampp\htdocs\desaganten\.env
2. Gunakan Notepad atau text editor lainnya
3. Cari bagian database dan ubah menjadi:

   DB_CONNECTION=mysql
   DB_HOST=127.0.0.1
   DB_PORT=3306
   DB_DATABASE=desaganten
   DB_USERNAME=root
   DB_PASSWORD=

4. Simpan file (Ctrl+S)

JALANKAN MIGRASI DATABASE:
1. Kembali ke Command Prompt
2. Pastikan masih di: C:\xampp\htdocs\desaganten
3. Ketik: php artisan migrate
4. Tekan Enter
5. Jika ditanya "Do you really wish to run this command?", ketik: yes
6. Tunggu hingga selesai

ISI DATABASE DENGAN DATA CONTOH:
1. Ketik: php artisan db:seed
2. Tekan Enter
3. Tunggu hingga selesai

12. TROUBLESHOOTING
================================================================================

MASALAH UMUM DAN SOLUSI:

A. XAMPP PORT 80 SUDAH DIGUNAKAN:
   Solusi:
   1. Buka XAMPP Control Panel
   2. Klik "Config" pada Apache
   3. Pilih "httpd.conf"
   4. Cari "Listen 80"
   5. Ubah menjadi "Listen 8080"
   6. Simpan file
   7. Restart Apache
   8. Akses dengan: http://localhost:8080

B. MYSQL PORT 3306 SUDAH DIGUNAKAN:
   Solusi:
   1. Tutup aplikasi yang menggunakan port 3306
   2. Atau restart komputer
   3. Jalankan ulang XAMPP

C. COMPOSER INSTALL GAGAL:
   Solusi:
   1. Ketik: composer clear-cache
   2. Ketik: composer install --ignore-platform-reqs
   3. Atau ketik: composer install --no-scripts

D. NPM INSTALL GAGAL:
   Solusi:
   1. Ketik: npm cache clean --force
   2. Hapus folder node_modules
   3. Ketik: npm install --force

E. DATABASE CONNECTION ERROR:
   Solusi:
   1. Pastikan MySQL di XAMPP berjalan
   2. Cek database 'desaganten' sudah dibuat
   3. Cek file .env sudah benar
   4. Restart MySQL di XAMPP

F. PERMISSION DENIED ERROR:
   Solusi:
   1. Tutup Command Prompt
   2. Buka Command Prompt sebagai Administrator
   3. Ulangi perintah yang error

G. PHP ARTISAN SERVE GAGAL:
   Solusi:
   1. Pastikan tidak ada aplikasi lain di port 8000
   2. Coba: php artisan serve --port=8001
   3. Akses dengan: http://localhost:8001

H. HALAMAN BLANK/WHITE SCREEN:
   Solusi:
   1. Ketik: php artisan config:clear
   2. Ketik: php artisan cache:clear
   3. Ketik: php artisan view:clear
   4. Refresh browser

I. PDF TIDAK BISA DOWNLOAD:
   Solusi:
   1. Ketik: composer require barryvdh/laravel-dompdf
   2. Ketik: php artisan vendor:publish --provider="Barryvdh\DomPDF\ServiceProvider"
   3. Restart server

üÜï J. 3D CHARTS TIDAK MUNCUL/ERROR:
   Solusi:
   1. Cek koneksi internet (Chart.js dari CDN)
   2. Clear browser cache (Ctrl+Shift+Delete)
   3. Pastikan JavaScript enabled di browser
   4. Coba browser lain (Chrome/Firefox recommended)
   5. Cek Console Error di Developer Tools (F12)

üÜï K. CHART.JS 3D PLUGIN ERROR:
   Solusi:
   1. Pastikan CDN Chart.js dan plugin 3D load correctly
   2. Check network tab di Developer Tools
   3. Gunakan offline Chart.js jika CDN bermasalah:
      - Download Chart.js ke public/js/
      - Update script src di blade files

üÜï L. STATISTIK PAGE SLOW LOADING:
   Solusi:
   1. Ketik: php artisan optimize
   2. Ketik: php artisan config:cache
   3. Ketik: php artisan route:cache
   4. Ketik: php artisan view:cache
   5. Consider memindah Chart.js ke local storage

üÜï M. MOBILE CHART 3D TIDAK RESPONSIVE:
   Solusi:
   1. Test di browser mobile mode (F12 > Device Mode)
   2. Pastikan viewport meta tag ada di layout
   3. Check CSS media queries untuk chart container
   4. Adjust chart options untuk mobile screen

üÜï N. SIDEBAR TIDAK KONSISTEN:
   Solusi:
   1. Pastikan semua halaman extends 'layouts.app'
   2. Check include partials (navbar, sidebar)
   3. Verify Alpine.js loaded correctly
   4. Clear browser cache dan reload
